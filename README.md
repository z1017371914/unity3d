4.
Unity有一些插件，可以用来更快捷地开发，其中有一些比较热门，比如：
做UI的：NGUI
做动画的：iTween
做状态机切换的：PlayMaker
做例子效果的：FX maker

这些插件，在官网的插件商店有卖，当然也可以找得到免费资源
